API

These four PHP scripts uniquely extract data from the database by Bacteria ID(s), as well as other parameters when specified, then returns results in the form of a JSON object.

getByBacteriaGP.php

To access data from the Growth Parameters table, simply click on the script link above, in the address box after getByBacteriaGP.php, add ?bid="id1,id2,id3" where the Bacteria IDs are within double quotations and separated by a comma. One Bacteria ID is sufficient, omit any commas. Capitalization is unnecessary.

Example:
getByBacteriaGP.php?bid="ed002,ed007"
OR
getByBacteriaGP.php?bid="ed002"

getByBacteriaEXPT.php

To access data from the Experiment table, simply click on the script link above, in the address box after getByBacteriaEXPT.php, add ?bid="id1,id2,id3" where the Bacteria IDs are within double quotations and separated by a comma. One Bacteria ID is sufficient, omit any commas. Capitalization is unnecessary.

Example:
getByBacteriaEXPT.php?bid="ed002,ed007"
OR
getByBacteriaEXPT.php?bid="ed002"

getByBacteria_MC_GP.php

To access data from the Growth Parameters table by Bacteria ID, as well as, Main Source and/or Compound, simply click on the script link above. In the address box after getByBacteria_MC_GP.php, add ?bid="id1,id2,id3"&ms="mainsource"&cp="compound" where the Bacteria IDs, Main Source and/or Compound are within their respective double quotations. One Bacteria ID is sufficient, if multiple, separate with commas. Capitalization is unnecessary.

Example:
getByBacteria_MC_GP.php?bid="ed002,ed007"&ms="carbon"&cp="propionate"
OR
getByBacteria_MC_GP.php?bid="ed002"&ms="carbon"&cp="propionate"
